# Inventar-Software – Jonas Krüger

## Beschreibung
Die **Inventar-Software** ist ein Webprojekt, mit dem du eine persönliche Inventarliste erstellen, verwalten und speichern kannst. Alle Daten werden lokal im **Browser über Local Storage** gespeichert, sodass keine Cloud oder Server nötig ist.  

**Features:**
- Gegenstände hinzufügen, bearbeiten und löschen  
- Anzeige von Name und Menge  
- Dark/Light Mode  
- Speicherung aller Items direkt im Browser (Local Storage)  
- Benutzerfreundliches Design für Desktop und Mobile  

---

## Installation
1. Repository herunterladen oder klonen:
```bash
git clone https://github.com/Jonaskar-64Bit/Inventar-Software
```
2. Gehe in den Projekt Ordner und öffne **index.html** in deinem Browser

## Nutzung
1. Auf **"Neuen Gegenstand hinzufügen"** klicken
2. **Name** und **Menge** des Items eingeben
3. Auf **"Hinzufügen"** klicken -> Das Item erscheint in der Liste
4. Mit dem **Stift-Symbol** ein Item bearbeiten
5. Mit dem **Mülleimer-Symbol** ein Item löschen
6. Dark/Light Mode oben rechts über das Icon wechseln
7. Alle änderungen werden automatisch im Browser gespeichert

## Technologien
- HTML 5
- CSS 3
- JavaScript (ES6)
- Local Storage für Speicherung der Daten
